Windows 7连接网络共享打印机

发表于 2018-03-01 | 分类于 计算机 ， win7

一、连接前的准备

1、本教程适用于两台Windows 7（32位）计算机之间的连接。Win7—XP以及Win7(64位)不适用。

2、 所有计算机都要设置密码，配置完成后，任何一台计算机更改了密码，共享打印机就会失效。

3、 记录下对外共享打印机的计算机各项信息：

　　★ 计算机名（右击“计算机”——属性）

　　★ 用户名（如administrator）

　　★ 密码（如123456），此计算机必须设置密码。

4、确保对外共享打印机的计算机已经开机（非休眠），并且能正常打印，并对外共享。

二、配置实例

1、配置情况介绍

本实例中，对外共享打印机的计算机名“GAO3-LS01”，用户名“administrator”，密码“123456”，打印机共享名为“Lenovo LJ3600D”

2、配置过程

【第一步】添加Windows凭据

★ 打开控制面板（开始—控制面板）

★ 选择“凭据管理器”。如果找不到，选择“用户账户和家庭安全”，再找找。

★ 添加登录凭据

★ 输入打印机所在计算机的“网络地址（计算机名）”、“用户名”、“密码”

★ 确定。

注：建议删除多余的Windows凭据

【第二步】查找网络中的“GAO3-LS01”计算机

双击桌面“网络”（或者桌面“计算机”—左侧列表中“网络”）

稍等片刻将自动搜索网络中的计算机

左图方框处显示搜索进度（长时间不开搜，右键–刷新）

搜索完毕后找到“GAO3-LS01”（如右下图）。

【第三步】登录“GAO3-LS01”计算机

双击“GAO3-LS01”，这时就可看到该电脑上已经共享的打印机“Lenovo LJ3600D”，如下图。

双击“Lenovo LJ3600D”，连接并安装打印机（左下图），出现“您是否信任此打印机？”，勾选“不再…”，单击“安装驱动程序”。

【第四步】设置为默认打印机

在新添加打印机窗口中，单击“打印机”菜单——“设置为默认打印机”。

【第五步】打印测试
